# 卒論用図表レビュー 改訂版

この文書は、`thesis_artifacts_adoption_review.md` の指摘を反映した後の図表レビューである。実験1は主分析、実験2は補助分析として扱い、候補は「専門家確認候補」、評価は「LLM支援付き著者確認評価」として表現する。

## 改訂後の総合判断

改訂後の図表は、卒論本文に使用可能である。特に以下の点を修正した。

- `dataset_summary_table` はA0/B0をサンプル件数ではなく最終集計値に差し替えた。
- 実験1の結果表を、データセット、U-Net出現、評価指標、要約表に分割した。
- baseline比較は、提案手法の単純勝利ではなく、候補集合の性質比較であることを図表・READMEに明記した。
- 実験2pipelineに、Top20×7、family-level deduplication、46 unique families、LLM-assisted author reviewを明示した。
- 実験2baselineは「lightweight fulltext text baseline」と表記し、厳密なBERT型全文埋め込み実験とは断定しない形に統一した。

## 日本語キャプション案と本文説明

### framework_overview

キャプション案:
「特許文書から課題文脈・解決手段文脈を抽出し、異分野間の専門家確認候補ランキングとLLM支援付き著者確認評価につなげる本研究の全体枠組み」

本文説明:
「図Xに、本研究の全体枠組みを示す。特許文書のタイトル・要約・請求項から課題文脈と解決手段文脈を抽出し、それぞれをベクトル化して異分野間の候補ランキングを作成する。その後、抽出された候補をLLM支援付き著者確認評価により確認し、専門家確認候補として解釈可能かを評価した。」

### solution_type_quadrant

キャプション案:
「課題文脈類似度と解決手段文脈類似度に基づく、近接解決策型と異質解決策型の概念的位置づけ」

本文説明:
「図Xは、課題文脈と解決手段文脈の近さに基づき、本研究の2つの実験を概念的に位置づけたものである。実験1はU-Netを対象とする近接解決策型の主分析、実験2は洗浄・除去系技術を対象とする異質解決策型の補助分析として扱う。」

注意:
図中の位置は実測スコアではなく、研究設計上の概念的位置づけである。

### method_pipeline

キャプション案:
「データ取得、前処理、文脈抽出、ベクトル化、ランキング、LLM支援付き著者確認評価までの提案手法の処理フロー」

## 注意すべき表現

| 避けるべき主張 | 推奨表現 |
|---|---|
| 技術移転を直接的に予見したという主張 | 専門家確認候補として抽出できるかを確認した |
| 実装・適用の可否を実証したという主張 | 解釈可能な専門家確認候補として整理した |
| 企業行動を予見したという主張 | 花王の洗浄・除去系特許から、半導体洗浄課題に接続しうる候補を補助分析として抽出した |
| baseline群への単純勝利という主張 | 候補集合の性質がbaselineと異なることを確認した |
| 厳密なBERT型全文埋め込み実験という表現 | 軽量全文テキストbaseline |
| LLM単独で妥当性を決めたという主張 | LLMは文脈整理を支援し、評価はLLM支援付き著者確認評価として行った |
