# 実験2 結果まとめ：洗浄・除去系技術を対象とした異質解決策型補助分析

## 1. 実験概要

| 項目 | 内容 |
|---|---|
| 実験の位置づけ | 補助分析（異質解決策型） |
| 候補元ドメイン | 花王の洗浄・除去系技術（非半導体系） |
| 対象ドメイン | 半導体洗浄（既知事例S1に基づく用途クラスタ） |
| 評価対象件数 | 46件のユニークファミリー |
| 評価方式 | LLM支援付き著者確認評価（後ろ向き評価） |
| 注意点 | 技術導入可能性・企業進出予測を示すものではない |

---

## 2. データセット構成

| 記号 | 内容 | 件数 |
|---|---|---|
| **S1** | 花王の半導体洗浄関連既知事例（参照集合） | 公開公報311件 / ファミリー23件 |
| **C1** | 花王の洗浄・除去系非半導体候補母集団 | 522件 |
| S0\_pre | クラスタ初出日前のS0比較集合 | 72,373件（行数） |
| S0\_pre\_filtered | 用途語フィルタ後のS0比較集合 | 10,786件（行数） |

### C1候補母集団の作成条件

1. S1と同一ファミリーを**除外**（既知事例の再発見を避ける）
2. 半導体直接語を含むものを**除外**（非半導体系候補に絞る）
3. `substrate` 単独の出現は除外せず，**確認用フラグ**として保持
4. 各S1クラスタの**クラスタ初出日以前**に存在するC1候補を対象とする

---

## 3. S1用途クラスタの定義と構成

S1（半導体洗浄の既知事例）を用途別に7クラスタに分類した。各クラスタ内の最早日を**クラスタ初出日**として設定し，それ以前のC1候補を抽出する時系列条件として使用した。

| クラスタ名 | 内容・特徴 |
|---|---|
| 接着剤洗浄 | 接着剤残渣の除去技術 |
| 回路基板洗浄 | プリント基板・回路基板の洗浄技術 |
| CMP後洗浄 | 化学的機械研磨（CMP）後の残渣除去技術 |
| フラックス・電子部品洗浄 | フラックス残渣・電子部品の洗浄技術 |
| フォトレジスト・樹脂除去 | フォトレジスト・樹脂残渣の除去技術 |
| **半導体基板洗浄** | シリコン基板・半導体基板の洗浄技術 |
| シリコンウェーハ洗浄 | シリコンウェーハ表面の洗浄技術 |

---

## 4. 候補抽出フロー

```
S1（23ファミリー）
  → 7用途クラスタに分類
  → 各クラスタのproblem_contextをBERT系モデルでベクトル化
  → クラスタ初出日以前のC1候補（522件）のproblem_contextもベクトル化
  → コサイン類似度でクラスタ別ランキング
  → 各クラスタ上位20件を抽出（7クラスタ × 上位20件）
  → ファミリー単位で重複統合
  → 46件のユニークファミリー
  → LLM支援付き著者確認評価
```

---

## 5. 評価ラベルの定義

| ラベル | 定義 |
|---|---|
| ◎（強い） | 半導体洗浄・基板洗浄・残渣除去等に**直接接続しうる強い候補** |
| ○（妥当） | 精密洗浄・電子部品・材料表面処理等，**周辺領域として接続しうる候補** |
| △（弱い） | 課題・効果は近いが，対象物・工程・解決手段の対応が**弱い候補** |
| ×（不適切） | 異質解決策候補としての対応が**読み取りにくい候補** |

---

## 6. 評価結果（全体集計）

46件のユニークファミリーに対するLLM支援付き著者確認評価の結果。

| ラベル | 意味 | **件数** | **割合** |
|---|---|---|---|
| ◎ | 強い候補 | **11件** | 24% |
| ○ | 妥当候補 | **5件** | 11% |
| △ | 弱い候補 | **12件** | 26% |
| × | 不適切候補 | **18件** | 39% |
| **◎+○** | **強い or 妥当な専門家確認候補** | **16件** | **35%** |
| ◎+○+△ | 広義の確認候補 | 28件 | 61% |

> **解釈：** 全46件中16件（35%）が強い or 妥当な専門家確認候補として解釈可能であった。一方，×が18件（39%）と多く，全体としてノイズは多い。対象物・汚染物・工程条件の違いにより，半導体洗浄課題との対応が読み取りにくい候補が混入しやすい。

### クラスタ別の考察

| クラスタ | ◎+○ | 評価の傾向と理由 |
|---|---|---|
| **半導体基板洗浄** | **6/6（100%）** | 対象（基板洗浄）と課題が半導体洗浄に近く，候補の対応関係を読み取りやすかった |
| 回路基板洗浄 | 2/3（67%） | 電子部品洗浄という共通点があり，対応が読みやすいものが多かった |
| フラックス・電子部品洗浄 | 2/5（40%） | 電子部品への接続候補は読めるが，工程の差が一部影響 |
| フォトレジスト・樹脂除去 | 2/6（33%） | 有機物除去の課題は近いが，工程固有性（レジスト工程）の影響あり |
| シリコンウェーハ洗浄 | 2/7（29%） | ウェーハ特有の洗浄条件が求められるため，汎用洗浄技術との対応が弱くなりやすい |
| 接着剤洗浄 | 2/7（29%） | 対象物（接着剤成分）や汚染物の種類が半導体洗浄と異なる候補が多かった |
| **CMP後洗浄** | **0/12（0%）** | CMP後洗浄は工程固有性が非常に強く，非半導体系洗浄技術から直接対応を読み取ることが困難だった |

## 8. 軽量全文テキストベースラインとの比較

| 指標 | 値 |
|---|---|
| **提案手法**（課題文脈ベース）件数 | 46件 |
| 軽量全文テキストベースライン件数 | 56件 |
| **重複ファミリー数** | **11件** |
| 提案手法のみに含まれる候補 | 35件 |
| ベースラインのみに含まれる候補 | 45件 |

### 重複11件の内訳

| ラベル | 件数 |
|---|---|
| ◎（強い） | 4件 |
| ○（妥当） | 3件 |
| △（弱い） | 2件 |
| ×（不適切） | 2件 |

> **解釈：** 重複がわずか11件（提案手法の24%，ベースラインの20%）にとどまった。これは課題文脈ベースのランキングと全文テキストベースのランキングが**異なる候補集合を提示する**ことを示す。全文テキストでは対象物・用途語・請求項の表現が類似度に強く影響するのに対し，提案手法では「何の課題に対応しているか」に焦点を当てるためである。
>
> ただし，**ベースラインのみの候補（45件）は未評価**であるため，この比較から提案手法の精度優位を結論づけない。本比較は候補集合の傾向差を確認する補助分析として位置づける。

## 9. まとめ

| 観点 | 結果 |
|---|---|
| 評価対象件数 | 46件のユニークファミリー |
| ◎+○（強い or 妥当な候補） | **16件（35%）** |
| ×（不適切候補） | **18件（39%）** |
| 最良クラスタ | 半導体基板洗浄：**◎+○ = 6/6（100%）** |
| 最困難クラスタ | CMP後洗浄：**◎+○ = 0/12（0%）** |
| 全文ベースラインとの重複 | 11件（双方の候補集合の2割程度） |

> **位置づけ：** 実験2は，対象物や工程が大きく異なる場合（異質解決策型）においても，課題文脈を手がかりに専門家確認候補を整理できる可能性を補助的に示した。一方で，候補の解釈可能性はクラスタ設計に大きく依存し，工程固有性が強いクラスタ（CMP後洗浄等）では有効な候補が得にくいことも確認された。本実験は花王の半導体分野への進出予測や技術導入可能性を示すものではない。
