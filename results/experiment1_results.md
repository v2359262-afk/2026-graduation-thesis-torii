# 実験1 結果まとめ：U-Netを対象とした近接解決策型候補抽出

## 1. 実験概要

| 項目 | 内容 |
|---|---|
| 実験の位置づけ | 主分析（近接解決策型） |
| 参照ドメイン（A0） | 医療画像解析（Medical Image Analysis） |
| 対象ドメイン（B0） | 製造欠陥検出（Manufacturing Defect Detection） |
| 評価対象手法 | U-Net（U-Net / UNet / nnU-Net / U-Net++ / UNet++） |
| 評価方式 | 後ろ向き評価（Retrospective Evaluation） |
| 過去期間（候補抽出） | 2015〜2018年 |
| 将来期間（後ろ向き確認） | 2019〜2024年 |

## 4. 候補抽出結果（U-Net後ろ向き確認）

| 項目 | 値 | 解釈 |
|---|---|---|
| **U-Netのランキング順位** | **2位** | 過去期間の具体的手法ランキングで上位に抽出された |
| 上位3件以内の抽出 | **該当** | 専門家確認候補として十分上位に含まれた |
| A0 過去期間 U-Net件数 | **55件** | 医療画像解析の過去期間（2015–2018）で確認 |
| B0 過去期間 U-Net件数 | **0件** | 製造欠陥検出の過去期間では未確認 |
| B0 将来期間 U-Net件数 | **314件** | 製造欠陥検出の将来期間（2019–2024）で出現・増加 |
| 評価の位置づけ | 後ろ向き評価 | 将来予測ではなく，既知の後年出現を用いた確認 |

> **注：** U-Netのランキング順位（2位）は，過去期間（2015–2018）の情報のみを使って生成したGapScoreランキングによる。ランキング生成時点では将来期間（2019–2024）のB0出現情報は使用していない。

## 5. ベースライン比較

### 5.1 提案手法 vs 全文類似上位ベースライン（主比較）

| 候補集合 | 件数 | 平均スコア | 妥当候補率 | ◎（強い） | ○（妥当） | △（弱い） | ×（不適切） |
|---|---|---|---|---|---|---|---|
| **提案手法（GapScore）** | **23件** | **1.61** | **52.2%** | **5件** | **7件** | 8件 | 3件 |
| 全文類似上位ベースライン | 30件 | 0.567 | 16.7% | 1件 | 4件 | 6件 | **19件** |

> **スコアの定義：** ◎=3点，○=2点，△=1点，×=0点。妥当候補率 = (◎+○) / 全体件数。

## 7. 評価ラベルの定義

| ラベル | 定義 |
|---|---|
| ◎（強い） | B0分野の課題に対し，解決手段として強く読み取れる候補 |
| ○（妥当） | B0分野の課題との対応が妥当な専門家確認候補 |
| △（弱い） | 課題は近いが解決手段の対応が弱い候補 |
| ×（不適切） | 専門家確認候補としての対応が読み取りにくい候補 |

---

## 8. まとめ

| 観点 | 結果 |
|---|---|
| U-Netのランキング順位 | **2位**（過去期間情報のみを使用） |
| 後ろ向き評価の対応 | 過去期間0件→将来期間314件（B0でのU-Net出現増加と対応） |
| 提案手法の妥当候補率 | **52.2%**（全文類似上位ベースラインの16.7%に対して高い） |
| 全文類似上位ベースラインの×率 | **63.3%**（不適切候補が多く混入） |
| 提案手法の×率 | **13.0%**（不適切候補が少ない） |

> **位置づけ：** 本実験は，医療画像解析から製造欠陥検出への技術移転の因果関係を示すものではない。課題文脈・解決手段文脈の分離に基づくスクリーニング手法が，専門家確認候補を説明しやすい形で抽出できることを後ろ向き評価により確認したものである。
